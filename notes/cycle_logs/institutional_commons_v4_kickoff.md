  - `regulated_fishery`
  - `community_irrigation`
  - `forest_co_management`
- governance-friction regimes:
  - `ideal`
  - `constrained`
- new friction outputs:
  - `missed_target_rate`
  - `targeted_share`
  - `delayed_intervention_count`
  - `governance_budget_spent`
- LLM strategy-bank tooling for Harvest
- LLM population governance-map script
- LLM turnover pilot script

The scenarios are treated as literature-backed institutional archetypes rather than field-calibrated replicas.
