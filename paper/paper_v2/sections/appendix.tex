\section{Appendix}

The appendix-only material serves three roles. For the Fishery Commons study, hard-tier runs remain useful as stress confirmation but are not used for causal ranking once collapse saturates across all governance conditions. For the Harvest Commons study, the earlier fixed-composition matrix is retained as a pilot precursor that motivated the invasion upgrade but is no longer treated as the main evidence. Finally, the full Harvest Stage B pilot matrix, Stage C high-power paired-delta tables, and confidence-interval summaries are maintained as supplementary artifacts so that every number in the text can be traced to a frozen output file.
