# `paper_v4_institutional_commons`

Working draft for the literature-backed institutional commons extension.

This paper keeps the project in the same line of work:
- commons
- adversarial pressure
- governance architecture

The extension adds:
- literature-backed scenario archetypes
- oversight-friction regimes
- LLM-generated Harvest strategy populations

The scenarios are treated as policy-interpretable archetypes, not field-calibrated reconstructions.

## Build

```bash
cd paper/paper_v4_institutional_commons
latexmk -pdf main.tex
```
